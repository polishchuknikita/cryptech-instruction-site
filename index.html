Настройки проекта

Перечень пунктов секции:

  • title - Текстовое поле, для редактирования ссылки проекта
    вводим сюда такое же название, которое у проекта на coinmarket (если он есть там)
  • isApiProject - Тумблер поле, для выбора брать информацию с апи или нет
  • tabsTitles - Интерактивное поле, для редактирования перечня табов, где должен отображаться проект
    Значение поля text, должно быть таким же, как и btnValue в кнопке таба на главной странице

Секция Информация о проекте

Перечень пунктов секции:

  • name - Текстовое поле, для редактирования названия проекта
  • logoSrc - Файловое поле, для редактирования логотипа проекта
  • subName - Текстовое поле, для редактирования аббревиатуры проекта
  • projectWithProgress - Тумблер поле, если нужно отображать прогресс проекта
  • company - Текстовое поле, для редактирования заголовка стоимости валюты
  • progressTitle - Текстовое поле, для редактирования заголовка времени изменения проекта
  • money - Текстовое поле, для редактирования стоимости валюты
    Вводим только число, без лишних символов (если десятичное число, то с точкой)
  • progressTitle - Текстовое поле, для редактирования времени изменения проекта
    Вводим только число, без лишних символов (если десятичное число, то с точкой)
  • projectTitle - Текстовое поле, для редактирования заголовка под логотипом
  • projectText - Текстовое поле, для редактирования текста под логотипом

Инструкции проекта

Перечень пунктов секции:

  • StakeSectionTitle - Текстовое поле, для редактирования заголовка секции
  • projectInstructionList - Интерактивное поле, для редактирования списка ссылок проекта
    • number - Текстовое поле, для редактирования порядкового номера инструкции
    • title - Текстовое поле, для редактирования заголовка инструкции

    Тут есть 2 дополнительных стиля, которые можно оформлять
    • Если нужна синяя плашка, которая будет ссылкой.
    • Если нужна синяя плашка, которая не будет ссылкой.
    • Если нужна красная плашка, которая будет ссылкой.
    • Если нужна красная плашка, которая не будет ссылкой.
